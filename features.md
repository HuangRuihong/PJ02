# Split-it-Smart 功能清單 (Feature List)

本系統是一個專為室友、死黨設計的**非同步分帳與債務管理系統**。透過模組化重構，我們提供了更清晰的功能劃分與更強的擴充性。

## 1. 個人帳務管理 (Personal Accounting) —— 角色 A 負責
*   **個人債務摘要**：一鍵查看自己目前「欠誰多少」與「誰欠我多少」。
*   **好友系統**：建立與管理信任的好友名單，作為分帳的基礎實體。
*   **QR 碼整合**：自動生成個人專屬的 ID QR 碼，方便快速識別與連結。
*   **個人隱私隔離**：確保個人資料與無關群組帳項物理隔離。

## 2. 群組協作與分帳 (Group & Split) —— 角色 B 負責
*   **多人群組建立**：透過唯一邀請碼快速建立或加入室友帳本。
*   **非同步提案機制**：
    *   消費發起後需經過參與者確認，確保每一分錢都經過認可。
    *   視覺化狀態追蹤：✅ 已確認、⏳ 待處理、❌ 已拒絕。
*   **彈性分帳算法**：
    *   **平均分攤**：一鍵平分。
    *   **自訂金額**：手動輸入每人具體負擔，滿足非對稱消費。
*   **智能餘額結算**：內建「分 (Integer)」精度處理，杜絕浮點數誤差。

## 3. 核心底層技術 (Technical Excellence)
*   **模組化 Service 架構**：
    *   `PersonalService` 與 `GroupService` 完全解耦，支持多人平行開發。
    *   `DebtSystem` 外觀模式整合，確保 UI 調用簡便。
*   **事務保護 (Atomicity)**：確保複雜的分帳寫入作業遵循「全有或全無」原則，不產生垃圾數據。
*   **SQLite 高效存儲**：單檔案資料庫，資料備份與遷移極其簡單。

## 4. 極簡開發與部署 (DevOps & UX)
*   **⚡️ 一鍵啟動 (`run.bat`)**：自動處理 Windows 編碼與 Python 環境檢查。
*   **🚀 一鍵 Git 設置 (`setup_git.bat`)**：引導開發者快速連結遠端 GitHub 倉庫。
*   **🤝 完整開發手冊**：內建 `Handover_Guide.md` 與 `DEVELOPMENT_GUIDE.md` 助開發者秒上手。
*   **現代化 GUI**：基於 `customtkinter` 打造，支援深色模式。
