# 專案資料夾架構說明 (Modularized Version)

本文件詳述 **Group Ledger (Split-it-Smart)** 專案的資料夾架構與各模組職責，供開發者維護與擴充參考。

## 📂 全域目錄架構

```text
group-ledger/ (專案根目錄)
├── README.md                           # 全域專案導引與架構說明
├── server/                             # 【伺服器端】FastAPI 應用程式
│   ├── api_server.py                   # 後端 API 服務主入口
│   ├── requirements.txt                # 伺服器端依賴套件清單
│   ├── .env                            # 伺服器環境配置 (HOST, PORT)
│   └── run_online_A.bat                # 伺服器啟動捷徑
└── client/                             # 【客戶端】Tkinter GUI 應用程式
    ├── .env                            # 客戶端環境配置 (API_BASE_URL)
    ├── README.md                       # 客戶端讀我文件
    ├── scripts/                        # 【啟動與入口】
    │   ├── main_gui.py                 # 主 GUI 核心與分頁導覽邏輯
    │   ├── run_app.py                  # 應用程式啟動入口
    │   ├── requirements.txt            # 客戶端依賴套件清單
    │   └── run_online_B.bat            # 客戶端聯網模式啟動腳本
    ├── src/                            # 【核心邏輯與組件】
    │   ├── analysis/                   # 數據統計與圖表視圖 (分析、日曆)
    │   ├── auth/                       # 使用者登入與權限介面
    │   ├── groups/                     # 群組管理、帳務分攤與結算邏輯
    │   ├── intelligence/               # 業務邏輯外觀層 (DebtSystem, NetworkFacade)
    │   ├── personal/                   # 個人帳務、好友管理與歷史流水
    │   └── shared/                     # 通用底層組件、模型與資料庫存取
    ├── doc/                            # 【文件與 SQL】開發紀錄與定義
    ├── tests/                          # 【測試】自動化單元測試驗證
    └── tools/                          # 【維護】系統營運與自動化腳本
```

## 🏗️ 詳細模組說明 (client/src)

### 1. intelligence/ (智慧邏輯層)
*   `__init__.py`: 模組初始化。
*   `debt_system.py`: 本地模式的 Facade 類別，整合所有 Service。
*   `network_facade.py`: **聯網模式代理**，將所有本地呼叫轉發至遠端 API，實現前後端分離。

### 2. groups/ (群組帳務)
*   `group_frame.py`: 群組動態介面 (支出清單、結算按鈕)。
*   `group_service.py`: 核心結算演算法 (一般結算、智慧抵銷演算法)。

### 3. shared/ (共享基礎設施)
*   `base_service.py`: SQLite 基礎連線與資料庫自動初始化。
*   `models.py`: 全域列舉 (TransactionStatus, Category) 與資料結構。
*   `dialogs.py`: 統一設計的彈窗組件 (新增交易、加入群組等)。
*   `data/`: 存放物理資料庫 `accounting.db` 與 `config.json`。

### 4. personal/ (個人中心)
*   `personal_frame.py`: 個人總覽與所有歷史交易流水分頁。
*   `friends_frame.py`: 好友清單與私下債務快速還款。
*   `personal_service.py`: 個人專屬帳務與好友邏輯處理。

### 5. analysis/ (數據視覺化)
*   `analysis_frame.py`: 圓餅圖與支出趨勢分析。
*   `calendar_frame.py`: 日曆視圖，標註每日支出。

### 6. auth/ (認證模組)
*   `auth_ui.py`: 登入介面與「記住帳號」邏輯。

## 📄 文件與資料庫管理 (client/doc)
*   `schema.sql`: 完整資料庫 DDL 定義，包含欄位註解。
*   `migrations.sql`: 紀錄資料庫結構的變更歷史。
*   `資料夾架構.md`: (本文件) 維護專案的整潔與可讀性。
